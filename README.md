# Raven-Storm-IT-Army-of-Ukraine
Gist of a Raven-Storm Session with IT Army of Ukraine DDoS Urls

# Getting Started 

### Install Raven Storm
```
curl -s https://raw.githubusercontent.com/Taguar258/Raven-Storm/master/install.sh | sudo bash -s
```
(See https://github.com/Taguar258/Raven-Storm)

### Run Raven-Storm
```
python3 main.py
```

### Save the below Raven-Storm Session file and Run it

```
load
```

Then enter path to your saved Session file and answer 'Y' when promted.

NOTE: Much more effective if run from within Russia via VPN/VPS.
